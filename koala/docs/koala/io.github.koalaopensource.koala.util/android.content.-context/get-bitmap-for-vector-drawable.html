getBitmapForVectorDrawable

fun Context.getBitmapForVectorDrawable(@DrawableRes id: Int, measurementPx: Int = -1, measurement: Measurement = Measurement.HEIGHT): Bitmap

Gets the bitmap object of the vector drawable resource scaled to the given measurement.

If measurementPx is provided, it is used to scale the vector drawable keeping it as width or height depending upon measurement.

Parameters

id - vector drawable resource to be used use

measurementPx - the size in pixels to scale to

measurement - the measurement type of measurementPx

Exceptions

Resources.NotFoundException - if the given resource does not exist

IllegalArgumentException - if the given resource id does not correspond to a vector drawable

Return
bitmap containing the vector drawable

Receiver
Context
